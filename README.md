# Specron example

Install the CLI.

```
$ npm install -g @specron/cli web3@1.0.0-beta.36 solc
```

Create a project.

```
$ mkdir myproject
$ cd myproject
$ specron init
$ npm install
$ npm test
```

Happy coding.
